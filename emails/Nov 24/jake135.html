ClickUpdates 3.01

Dive into the latest platform updates
 
Last week marked a significant milestone with the launch of ClickUp 3.0. In upcoming ClickUpdates emails, we'll be digging into the key new features of the platform.
 
 
See All New Features
 
 
Global Action Toolbar
 
ClickUp's new Toolbar centralizes primary actions in your Workspace, offering quick access to task creation, document management, Universal Search, ClickUp AI, and more! 
 
 
Find Out More
 
All New Sidebar
 
We've completely rebuilt the sidebar to be more performant and optimize how you navigate/organize your Workspace. It also provides centralized access to Docs, Whiteboards, and Dashboards in brand-new Hubs.
 
 
Find Out More
 
Task View 3.0
 
Task view has been redesigned to help you see information side by side. Subtasks are also super easy to navigate with the Task view sidebar, and Task properties like Custom Fields and action items are easily accessible.
 
 
Find Out More
 
Inbox
 
The new Inbox is fast, refreshless, reliable, and instantly syncs across devices and browsers. It's a complete overhaul of how you collaborate with your team and keep work moving.
 
 
Find Out More
 
Admin Settings
 
With ClickUp 3.0, we made sure you have the tools needed to roll it out reliably for your team. The new Admin Settings allow you to set the default version of ClickUp for your Workspace.
